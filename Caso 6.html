Transfer Learning con MobileNetV2

¿Por qué Transfer Learning?

🚀 Entrenamiento Más Rápido: En lugar de entrenar desde cero (días), transfer learning logra resultados en horas.
📊 Menos Datos Requeridos: Aprovecha conocimiento de millones de imágenes de ImageNet, requiere menos ejemplos por clase.
🎯 Mejor Precisión: Características pre-aprendidas (bordes, texturas) son transferibles a hojas de plantas.
📱 Optimizado para Móviles: MobileNetV2 está diseñado para dispositivos con recursos limitados (smartphones).

Estrategia de Entrenamiento en Dos Fases

1
Fase Inicial: Base Congelada

  • ✅ Capas de MobileNetV2: Congeladas
  • ✅ Solo entrenar nuevas capas superiores
  • ✅ Learning Rate: 0.001 (Adam)
  • ✅ Épocas: 5
  • ✅ Objetivo: Adaptar clasificador a plantas

2
Fase Fine-Tuning: Ajuste Fino

  • ✅ Últimas 50% capas: Descongeladas
  • ✅ Primeras 50% capas: Congeladas
  • ✅ Learning Rate: 0.0001 (10x menor)
  • ✅ Épocas: 5 adicionales
  • ✅ Objetivo: Ajuste fino de características

Métricas Críticas para Agricultura

¿Por Qué Importan Falsos Positivos y Falsos Negativos?

❌ Falsos Positivos (FP): El modelo dice que hay enfermedad cuando la planta está sana.
→ Consecuencia: Aplicación innecesaria de pesticidas, costos adicionales, impacto ambiental
⚠️ Falsos Negativos (FN): El modelo dice que está sana cuando hay enfermedad.
→ Consecuencia: MÁS GRAVE - Enfermedad no tratada se propaga, pérdida total de cultivo
✅ Verdaderos Positivos (TP): Detección correcta de enfermedad.
→ Tratamiento oportuno salva el cultivo

Optimización de Recall para Enfermedades Graves

Prioridad al Recall: Es mejor detectar todas las enfermedades (incluso con algunos falsos positivos) que perder enfermedades reales.
Umbrales Ajustables: En producción, se puede bajar el umbral de confianza para enfermedades devastadoras como el tizón.
Análisis de Probabilidades: El modelo provee probabilidades para cada clase, permitiendo alertas de "revisar manualmente" en casos ambiguos.

Código de Implementación

# Data Augmentation para imágenes de plantas datagen = ImageDataGenerator( rotation_range=20, width_shift_range=0.2, height_shift_range=0.2, horizontal_flip=True, zoom_range=0.2, validation_split=0.2 ) # Transfer Learning con MobileNetV2 base_model = keras.applications.MobileNetV2( weights='imagenet', include_top=False, input_shape=(256, 256, 3) ) # Fase 1: Entrenar solo capas superiores base_model.trainable = False model = keras.Sequential([ base_model, keras.layers.GlobalAveragePooling2D(), keras.layers.Dense(128, activation='relu'), keras.layers.Dropout(0.3), keras.layers.Dense(38, activation='softmax') ]) model.compile(optimizer='adam', loss='sparse_categorical_crossentropy', metrics=['accuracy']) # Entrenar Fase 1 history1 = model.fit(datagen.flow(...), epochs=5) # Fase 2: Fine-tuning base_model.trainable = True fine_tune_at = len(base_model.layers) // 2 for layer in base_model.layers[:fine_tune_at]: layer.trainable = False model.compile( optimizer=keras.optimizers.Adam(lr=0.0001), loss='sparse_categorical_crossentropy', metrics=['accuracy'] ) # Entrenar Fase 2 history2 = model.fit(datagen.flow(...), epochs=5)

Limitaciones y Consideraciones Éticas

Desafíos en el Mundo Real

🌦️ Condiciones Variables: El modelo entrenado con fotos controladas puede fallar con lluvia, poca luz o fondos complejos.
🌍 Generalización Geográfica: Enfermedades pueden verse diferentes en distintos climas o variedades de cultivos.
🆕 Nuevas Enfermedades: El modelo no puede detectar enfermedades emergentes no vistas en entrenamiento.
📚 Brecha Digital: Agricultores sin smartphones o alfabetización digital pueden quedar excluidos.
💰 Dependencia Tecnológica: Riesgo de que soluciones comerciales reemplacen conocimiento tradicional agrícola.

Recomendaciones Éticas

🤝 Herramienta de Apoyo: El modelo debe asistir, no reemplazar, al conocimiento del agricultor.
🆓 Acceso Abierto: Priorizar soluciones open-source para que sea accesible en países en desarrollo.
📖 Educación Paralela: Combinar IA con programas de capacitación agrícola.
🔍 Transparencia: Explicar al usuario el nivel de confianza y cuándo consultar a un experto.
